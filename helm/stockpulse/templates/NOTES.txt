StockPulse 배포 완료!

== 접근 방법 ==

1. Ingress 주소 확인:
   kubectl get ingress -n {{ .Values.global.namespace | default .Release.Namespace }}

2. API Gateway 헬스 체크:
   curl http://{{ .Values.ingress.host }}/api/v1/health

3. 실시간 WebSocket:
   ws://{{ .Values.ingress.host }}/ws/realtime

== 파드 상태 확인 ==
kubectl get pods -n {{ .Values.global.namespace | default .Release.Namespace }}

== HPA 상태 확인 ==
kubectl get hpa -n {{ .Values.global.namespace | default .Release.Namespace }}

== 로그 확인 ==
kubectl logs -n {{ .Values.global.namespace | default .Release.Namespace }} deploy/api-gateway --tail=50

== Grafana (관측) ==
kubectl port-forward -n {{ .Values.global.namespace | default .Release.Namespace }} svc/grafana 3000:80
http://localhost:3000
