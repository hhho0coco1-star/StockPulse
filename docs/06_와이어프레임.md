# StockPulse 와이어프레임

> **모바일 우선** 설계. 앱(React Native)과 모바일 웹이 화면·UX를 공유하고, 데스크톱 웹은 동일 화면의 반응형 확장이다. HTML 프로토타입(핵심 3화면)은 [wireframes/prototype.html](wireframes/prototype.html)을 브라우저로 열어 확인.

색 관례(국내): **상승=빨강, 하락=파랑**. 하단 5탭: 홈 · 인사이트 · 모의투자 · 커뮤니티 · 랭킹.

---

## 화면 목록 (전 7화면)

### 1. 로그인 / 회원가입
- 이메일·비밀번호 + 소셜 로그인(구글/카카오) 버튼
- 회원가입: 이메일 인증 → 닉네임
- API: `POST /auth/login`, `/auth/signup`, `GET /auth/oauth2/{provider}`

### 2. 메인 대시보드 ★(HTML 프로토타입)
- **내 워치리스트** 시세 리스트(종목명·코드·현재가·등락률, 실시간 갱신)
- **오늘의 강세 인사이트** 카드 리스트(점수 배지 + 근거 3줄)
- API: `GET /watchlist`, `GET /market/quotes`, `GET /insights/strong`
- 실시간: `/topic/market/{symbol}` 구독으로 시세 갱신

### 3. 종목 상세 ★(HTML 프로토타입)
- 상단: 현재가·등락, 시세 **차트**(TradingView)
- 탭: **차트 / 인사이트 / 뉴스 / 토론**
- **투자 전망 카드**: 종합 점수 + 모멘텀·실적·뉴스 분해(펼치기), 면책 문구
- 하단 고정: **모의 매수 / 모의 매도** 버튼
- API: `GET /market/candles`, `GET /insights/{symbol}/factors`
- 실시간: `/topic/market/{symbol}`, `/topic/insight/{symbol}`

### 4. 종목 커뮤니티 / 채팅
- 게시글 목록(좋아요·댓글 수) + 작성
- **실시간 채팅** 영역(종목별 토론방)
- API: `GET/POST /community/{symbol}/posts`, 댓글·좋아요
- 실시간: `/topic/chat/{symbol}` 구독, `/app/chat/{symbol}` 전송

### 5. 모의투자 ★(HTML 프로토타입)
- KPI: 총 평가금액·총 수익률·예수금·랭킹
- **보유 종목** 리스트(수량·평단·평가손익)
- **주문 패널**: 매수/매도 토글, 종목·유형(시장가/지정가)·수량 → 주문
- 주문 접수(202) → Saga 체결 → `/user/queue/orders`로 결과 통지
- API: `GET /portfolio`, `/account`, `POST /orders`

### 6. 랭킹
- 기간 탭(일간/주간/전체) + 수익률 리더보드 + 내 순위 강조
- API: `GET /ranking`, `/ranking/me`

### 7. 알림 설정
- 알림 규칙 목록·생성(목표가/급등락/뉴스), 워치리스트 종목 대상
- 디바이스 푸시 토큰 등록(FCM), 받은 알림 이력
- API: `GET/POST /alerts`, `POST /devices`, `GET /notifications`

## 웹 / 앱 차이 (1벌 설계 + 반응형)

| 구분 | 모바일(앱 · 모바일 웹) | 데스크톱 웹 |
|------|------------------------|-------------|
| 내비게이션 | 하단 5탭 바 | 좌측 사이드바 또는 상단 메뉴 |
| 레이아웃 | 1단(세로 스크롤) | 2~3단(시세·차트·커뮤니티 동시 표시) |
| 차트 | 전체폭 | 더 크게 + 보조지표 패널 |
| 푸시 | FCM 네이티브 | 웹 푸시(선택) |

> 화면 구성·컴포넌트·API는 동일. 데스크톱은 같은 컴포넌트를 더 넓은 그리드에 배치할 뿐이라 별도 와이어프레임을 만들지 않는다.

## HTML 프로토타입 범위

[wireframes/prototype.html](wireframes/prototype.html)에 **① 메인 대시보드 · ② 종목 상세 · ③ 모의투자** 3화면을 모바일 폭 목업으로 구현(자체 포함, 인터넷 불필요). 나머지 4화면(로그인·커뮤니티·랭킹·알림)은 위 설명으로 대체.
