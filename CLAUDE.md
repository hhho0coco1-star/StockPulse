# StockPulse

학습/포트폴리오용 실시간 투자 인사이트·모의투자·종목 커뮤니티 플랫폼 (Kotlin+Spring Boot MSA, 서비스 13개). 상세 설계는 `docs/01~12`, 작업 추적은 `todo.md` 참고.

## 하네스: StockPulse MSA 개발

**목표:** WBS 작업을 스펙→구현→검증 흐름으로 완료하는 MSA 백엔드 개발 자동화.

**트리거:** StockPulse 개발/구현/기능 추가/WBS 진행 관련 작업 요청 시 `stockpulse-dev` 스킬(오케스트레이터)을 사용하라. 구현 수정·부분 재실행·검증 다시 등 후속 작업도 동일. 단순 질문·설계 상담은 직접 응답 가능.

**팀:** architect(설계·문서, opus) / backend-dev(구현, sonnet) / qa(검증, opus). 정의는 `.claude/agents/`, 스킬은 `.claude/skills/`에서 관리.
